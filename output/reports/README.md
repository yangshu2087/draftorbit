# DraftOrbit Reports

Human-readable generated reports belong here and may be tracked when they are small Markdown evidence files.

- `audit/`: audit reports.
- `billing/`: billing, Stripe, and payment validation reports.
- `uat-full/`: full UAT, benchmark, and evidence-index Markdown reports.
- `hermes/`: Hermes-generated DraftOrbit reports when explicitly requested.

Raw screenshots, browser JSON, runtime visual assets, and large provider artifacts belong in local-only `output/playwright/` or `artifacts/`, not in tracked report docs.
