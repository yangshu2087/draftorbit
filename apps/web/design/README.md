# Web Design Inputs

Use this directory for checked-in front-end design intent.

## Recommended contents

- `design-system.md` — implementation guardrails for layout, typography, states, and responsive behavior
- `figma-links.md` — Figma files and node references
- `reference-shots/` — screenshots or cropped references
- `stitch/` — Google Stitch inputs or exports
- `ai-studio/` — Google AI Studio prompts or prototype exports

## Working rules

- Prefer checked-in design inputs over chat-only descriptions.
- Store screenshots, Figma references, design notes, and prototype exports here before major implementation work when possible.
- Keep raw exports in subfolders and keep human guidance concise.
- If you borrow inspiration from public references, note the chosen references and the translated local rules here or in `../../docs/agent-handoff.md`.
