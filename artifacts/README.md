# DraftOrbit Artifacts

Machine-readable outputs, screenshots, benchmark data, and runtime visual assets belong here.

Recovery policy:

- Keep this directory locally as evidence for restored runs and visual/source pipeline validation.
- Do not commit large runtime files by default; `.gitignore` tracks only this README unless a future task explicitly approves a specific evidence bundle.
- Do not store secrets or `.env` files here.

Related report indexes live under `output/reports/`.
