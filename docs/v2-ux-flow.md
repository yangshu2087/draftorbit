# DraftOrbit V2 体验流（用户视角）

## 主路径（无自由文本）
1. 进入 `/chat`
2. 选择目标、受众、语气、内容类型
3. 一键生成（显示步骤进度与耗时）
4. 查看结果包（主版本 + 变体 + 配图建议）
5. 质量门控通过后提交发布
6. 在发布队列查看状态回写

## 可选增强路径
- 连接 Obsidian / 本地文件 / URL 作为学习源
- 绑定多个 X 账号并设置默认账号
- 重建 Style DNA 后重新生成

## 错误处理规范
- 页面必须具备 `loading / empty / error / ready` 四态
- 错误卡片必须提供：可读原因、requestId、重试按钮
- 关键链路失败时给出下一步建议（改稿/重试/人工发布）
