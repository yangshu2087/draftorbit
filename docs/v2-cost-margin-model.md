# DraftOrbit V2 成本与毛利模型（USD）

## 1. 收入口径
- 套餐：Starter `$19` / Growth `$49` / Max `$99`
- 年付：8 折（保持现有策略）
- 超额包：后续启用（本轮保留接口位）

## 2. 成本口径
- 变量成本：LLM 请求成本（OpenRouter）+ 支付费率（Stripe）
- 固定成本：基础设施（API/Web/Worker/DB/Redis/日志）

## 3. 路由策略与成本护栏
- Trial：优先高阶模型，设置试用预算上限
- Paid：免费优先，低价兜底，质量不足时有限升档
- 每请求携带 `max_price`，预算接近阈值时自动降档

## 4. 毛利守门
- 日级预警：连续 3 天预测毛利低于 70% 触发保守路由
- 监控指标：
  - 免费命中率
  - 升档率
  - fallback 率
  - 平均单次成本
  - 质量分
